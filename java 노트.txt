클래스 tv 설계도, 객체 tv
객체 - 인스턴스

객체는 속성과 기능으로 이루어져있다.
속성 멤버변수 특성 필드 상태 (정적인것들)
기능 메서드 함수 행위 (동적인것들)

printf(서식) = 아직 안함.;

데이터 저장형태의 발전과정
변수 -> 배열(정수) -> 구조체 -> 클래스(함수 추가됨)

1. 캡슐화
2. 상속
3. 추상화
4. 다형성

public : 어디서나 접근 가능
protected : 상속에서만 접근 가능
default : 같은 패키지 내에서만 접근 가능 - 아무것도 안적혀있으면 기본으로 디폴트
private : 외부에서는 접근 X 내부 클래스에서만 사용가능

멤버변수에는 private 을 붙인다.
메소드에는 public을 붙이는게 일반적

static void func(){} : static 키워드가 붙어있는 메소드는 객체 생성하지 않고 접근 가능

void func(){} : static 없는 메소드는 반드시 객체 생성하고 접근
